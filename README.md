Unity Wedding Organizer

Aplikasi Wedding Organizer berbasis web yang dibangun menggunakan PHP Native (Tanpa Framework) dan MySQL. Project ini memiliki fitur untuk pelanggan (katalog, keranjang, pemesanan) dan halaman admin untuk mengelola pesanan.

Fitur Utama
Pelanggan:
- Melihat katalog paket pernikahan & detailnya.
- Menambahkan paket ke keranjang belanja.
- Checkout dan input detail acara.
- Melihat riwayat pesanan & status pembayaran.
- Cetak Nota Transaksi (PDF).

Admin:
- Dashboard ringkasan.
- Kelola Pesanan.
- Kelola Produk/Layanan (Tambah, Edit, Hapus).
- Cetak Laporan & Nota Admin.

Teknologi
- PHP Native
- MySQL / MariaDB
- HTML5 & CSS3
- JavaScript
